# TravelWitch

2D 플랫포머 멀티플레이 PVP 게임입니다.





### 빌드
```bash
# Unity 에디터로 .sln 파일 실행
```

### 실행
```bash
# Unity 에디터로 게임 실행
# 또는 별도의 구글 드라이브 실행 파일을 압축 해제 후 2dgame.exe를 실행
```



## 조작법
```
- **AD**: X축 이동
- **W**: 점프
- 점프 중 **W**: 2단점프
- **S**: 낙하
- **Shift**: 달리기
- **마우스**: 에임 조준
- *Left Click : 에임으로 총알 발사
- **Space**: 스킬 배리어 사용
- **ESC**: 종료
```
